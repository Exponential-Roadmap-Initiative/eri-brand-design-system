# ERI Brand Design System — Design Ideas

## Context
This is a **reference documentation site**, not a marketing page. The design must serve the content: developers and marketers need to scan, copy, and reference tokens, components, and guidelines quickly. The existing page at eri-expframework-beta-platform.manus.space/brand-design-system is the canonical reference — we mirror it exactly.

## Idea A — Faithful Documentation Mirror (Probability: 0.06)

**Design Movement:** Swiss International Typographic Style meets modern developer documentation
**Core Principles:**
1. Content-first — every design decision serves legibility and reference speed
2. Faithful replication — exact visual match to the existing page
3. Clean separation — sections are clearly delineated without decorative noise
4. Token-driven — all colours, spacing, and type come directly from the ERI design tokens

**Color Philosophy:** ERI brand green (#3ba559) as the sole accent. Off-white (#F9FAFB) page background. Near-black (#232323) for headings. Gray scale for body and muted text. No decorative colour.

**Layout Paradigm:** Single-column content flow with max-w-6xl container. Sections separated by generous vertical spacing (mb-16). Cards used for grouped content. No sidebar — this is a scrollable reference document.

**Signature Elements:**
- Green underline active states on interactive elements
- Code snippets in dark (#1a1a1a) cards with monospace font
- Colour swatches with copy-to-clipboard functionality

**Interaction Philosophy:** Minimal. Copy-to-clipboard on colour swatches. Smooth scroll. No animations that distract from content.

**Animation:** Hover transitions only — `transition-colors duration-150`. No entrance animations.

**Typography System:** Archivo (800 ExtraBold) for all headings. Archivo (400) for body. System monospace for code. Exactly as specified in the ERI brand guidelines.
